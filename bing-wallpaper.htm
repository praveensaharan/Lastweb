    Install Bing Wallpaper in 6 easy steps

    1. Download the Bing Wallpaper app
    2. Run BingWallpaper.exe
    3. Click Install now
    4. Click Finish
    5. If prompted from your browser, add the extension**
    6. Explore the Bing Wallpaper app in the system tray
